# taxation-without-representation

Research and analysis tools for government accountability and transparency.

## Projects

### [Congressional Voting Networks](/congressional_voting_networks/)
Network analysis of US Congress voting records from 1789 to present. Models relationships between legislators using graph theory to reveal bipartisanship, party cohesion, ideological shifts, and temporal evolution of congressional dynamics.

### [Judicial Integrity Analysis](/judicial_integrity_analysis/)
Comprehensive analysis of state and federal judicial integrity covering corruption, ethics, bias, and partisanship. Proceeds state by state, starting with Arizona (Phase 1). Uses public court data, performance reviews, sentencing statistics, and LLM-assisted auditing.
